				     Configuração do CSS

background-color: - cor do fundo do Bloco  

color - cor da fonte do Bloco 

border: - Alterações na Borda do Bloco

border-radius - Arredondar a Borda do Bloco

border-color - Alteração da cor da borda do Bloco

border-style - Alteração do estilo da borda

border-widht - Altera o comprimento da Borda

height - Alteração na Altura do Bloco (Altura = vertical)

width: - Alteranção no Comprimento do Bloco (comprimento = horizontal   )

cursor: pointer; - (Alteração na seta do mouse): (para mao ao passar em cima do button)

vertical-aling: top; - Alinhamento no sentido vertical

margin - // Da expaço entre os itens

	marging-right - margem para a direita
	marging-left - margem para a esquerda
	marging-top - margem para cima
	marging-bottom - margem para baixo


padding - // espaçamento dentro do botão

	padding-left - espaçamento para a esquerda
	padding-right - espaçamento para do direita
	padding-top - espaçamento para cima
	padding-bottom - espaçamento para baixo


transitions: - é um poderoso recurso para a criação de efeitos de animação nos elementos HTML
	(Ex: tem como adicionar mais de um elemento. 
	      transition: background-color 1s,
		color 1s;                          )

box-shadow: 0 0 0 black;(0 - o primeiro valor determina aonde vai ficar na horizontal 10px = direita
			 0 - o segundo valor determina aonde vai ficar na vertical 10px = desce
			 0 - o terceiro valor dertemina o sombreamento


object-fit - cover - a imagem mantém sua proporção de aspecto e preenche a dimensão dada. A imagem será recortada para caber
						 contain - a imagem, ela mantém sua proporção, mas é redimensionada para caber na dimensão fornecida:
						 none - a imagem não será redimensionada:
						 fill - a imagem é redimensionada para preencher a dimensão dada. Se necessário, a imagem será esticada ou comprimida para caber:
						 scale-down - a imagem será reduzida para a menor versão de noneou contain

object-position: propriedade é usada junto com object-fit para especificar como um <img> ou <video> deve ser
								 posicionado com coordenadas Direita/Esquerda dentro de sua "própria caixa de conteúdo".

				FORMATAÇÂO DE TEXTO CSS


font-style:  normal - The browser displays a normal font style. This is default	
             italic - The browser displays an italic font style	
             oblique - The browser displays an oblique font style	
             initial - Sets this property to its default value. Read about initial	
             inherit - Inherits this property from its parent element. Read about inherit


font-family: Altera a fonte das letras


font - Alteração na fonte


font-wight: bold; - deixa a fonte mais forte


font-size: - tamanho que a fonte vai ficar


text-aling : left	Aligns the text to the left	
             right	Aligns the text to the right	
             center	Centers the text	
             justify    Stretches the lines so that each line has equal width (like in newspapers and 
                        magazines)	
             initial	Sets this property to its default value. Read about initial	
             inherit	Inherits this property from its parent element. Read about inherit

line-height - The line-height property specifies the height of a line. / 
	      A line-heightpropriedade especifica a altura de uma linha.

text-decoration - overline
		  						line-through
		  						underline
		  						underline overline


display: propriedade especifica o comportamento de exibição
				 (o tipo de caixa de renderização) de um elemento.
								
								Classes:
								
inline; - Exibe um elemento como um elemento inline (como <span>).
Quaisquer propriedades de altura e largura não terão efeito. Este é o padrão.

block; - Exibe um elemento como um elemento de bloco (como <p>).
Ele começa em uma nova linha e ocupa toda a largura

contents; - Faz o contêiner desaparecer, tornando os elementos filhos filhos do elemento no próximo nível acima no DOM

flex; - Exibe um elemento como um contêiner flex de nível de bloco

grid; - Exibe um elemento como um contêiner de grade de nível de bloco

inline-block; - Exibe um elemento como um contêiner de bloco de nível inline.
O elemento em si é formatado como um elemento inline, mas você pode aplicar valores de altura e largura

inline-flex; - Exibe um elemento como um contêiner flex de nível inline

inline-grid; - Exibe um elemento como um contêiner de grade de nível inline

inline-table; - O elemento é exibido como uma tabela de nível inline

list-item; - Deixe o elemento se comportar como um elemento <li>

run-in; -  Exibe um elemento como bloco ou inline, dependendo do contexto

table; - Deixe o elemento se comportar como um elemento <table>

table-caption; - Deixe o elemento se comportar como um elemento <caption>

table-column-group; - Deixe o elemento se comportar como um elemento <colgroup>

table-header-group; - Deixe o elemento se comportar como um elemento <thead>

table-footer-group; - Deixe o elemento se comportar como um elemento <tfoot>

table-row-group; - Deixe o elemento se comportar como um elemento <tbody>

table-cell; - Deixe o elemento se comportar como um elemento <td>

table-column; - Deixe o elemento se comportar como um elemento <col>

table-row; - Deixe o elemento se comportar como um elemento <tr>

none; - O elemento é completamente removido

initial; - Define esta propriedade para seu valor padrão. Leia sobre initial

inherit; - Herda esta propriedade de seu elemento pai. Leia sobre inherit
